# Osakesäästäjät

This was my first piece of code that I received payment for. It was published as a listing in a Mikrobitti magazine in March 1985.

The listing is for an unexpanded Commodore VIC-20.

## Branches

| Branch | Description |
|--------|-------------|
| `main` | Original VIC-20 source (PRG + BASIC listing) |
| [`python`](../../tree/python) | Python reimplementation for terminal |
| [`ios`](../../tree/ios) | SwiftUI iOS app with Game Center leaderboards |
